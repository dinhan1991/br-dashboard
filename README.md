## Local Development

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
streamlit run app.py
```

## Deployment

Deployed on Streamlit Community Cloud.

**Note:** Database will reset on app restart. Upload your Excel files each time to populate data.

## Usage

1. Upload Buy Ready Report (Excel file)
2. Upload Drop Report (Excel file)
3. View and manage data
4. Export to Excel
